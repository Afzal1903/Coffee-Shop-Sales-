# 📊 Excel Sales Dashboard Project

## 📌 Overview

This project showcases an interactive Excel dashboard built using real-world transactional data. The goal of this project is to analyze sales performance, identify trends, and present insights in a visually intuitive way.

The dashboard allows users to explore data dynamically using filters and pivot-based visualizations.

---

## 🗂️ Dataset Description

The dataset contains transactional-level data with the following key fields:

* Transaction ID
* Transaction Date & Time
* Product Details
* Quantity Sold
* Sales Amount
* Hour, Day, and Month breakdown

---

## ⚙️ Features & Functionality

### 🔹 Data Cleaning

* Structured raw transactional data
* Extracted time-based insights (Hour, Day, Month)

### 🔹 Pivot Tables

* Aggregated sales performance
* Time-based analysis (daily, monthly trends)

### 🔹 Interactive Dashboard

* Dynamic filtering using slicers
* Visual insights for:

  * Sales trends
  * Peak hours
  * Product performance

---

## 📈 Key Insights

* Identifies peak sales hours and days
* Tracks monthly sales patterns
* Highlights top-performing products

---

## 🛠️ Tools Used

* Microsoft Excel

  * Pivot Tables
  * Slicers
  * Dashboard Design

---

## 🚀 How to Use

1. Download the Excel file from this repository
2. Open in Microsoft Excel (recommended)
3. Use slicers and filters to explore the dashboard
